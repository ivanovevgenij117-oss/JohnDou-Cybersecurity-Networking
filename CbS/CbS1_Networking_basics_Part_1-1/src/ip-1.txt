Рассчитать бинарную нотацию для адресов:
178.101.89.7
178 = 128 + 32 + 16 + 2 = 10110010
101 = 64 + 32 + 4 + 1 = 01100101
89 = 64 + 16 + 8 + 1 = 01011001
7 = 4 + 2 + 1 = 00000111
10110010.01100101.01011001.00000111


201.57.153.161
201 = 128+64+8+1 = 11001001
57 = 32+16+8+1 = 00111001
153 = 128+16+8+1 = 10011001
161 = 128+32+1 = 10100001
11001001.00111001.10011001.10100001

маска подстети на 3 машины:
255.255.255.248 или /29
нужно 3+2 адреса, то есть 5. ближайший бит - 8 - 11111000

маска подстети на 16 машин:
255.255.255.224 или /27
нужно 16+2 адреса, то есть 18. ближайший бит - 32 - 11100000

маска подстети на 32 машины:
255.255.255.192 или /26
нужно 32+2 адреса, то есть 34. ближайший бит - 64 - 11000000
